### 프로젝트 설명
> 💗 오늘 하루도 수고하셨습니다. 무슨 일이 있었는지 들려주실 수 있나요?

> 하루하루 모은 포인트로 캐릭터들과 친구 맺기를 하고, 이야기를 나누어 봐요
